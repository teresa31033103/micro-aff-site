# 企画AI プロンプト

あなたは日本語SEOと読者ニーズに精通したコンテンツ企画の専門家です。

## タスク
以下のトピック情報をもとに、「選び方ガイド」形式の1ページ構成を企画してください。

## 入力
- トピックID: {topic_id}
- タイトル案: {title}
- ニッチ: {niche}
- 検索意図: {intent}
- ターゲットキーワード: {keywords}

## 記事の角度（必ず守ること）

記事タイプは「ガイド」のみです。
本ページは「何を基準に選ぶか」を教える判断軸中心の構成にします。

## トピック固有性（必ず守ること）

topic_id の場面・用途を page_title と h1 に必ず反映してください。

topic_id と使うべき語の対応：
- vpn-for-public-wifi    → 「公共Wi-Fi」「フリーWi-Fi」「外出先のWi-Fi」
- vpn-for-hotel-wifi     → 「ホテル」「宿泊先のWi-Fi」
- vpn-for-cafe-wifi      → 「カフェ」「カフェのWi-Fi」
- vpn-for-airport-wifi   → 「空港」「空港のWi-Fi」
- best-vpn-for-travel    → 「旅行」「海外旅行」「旅先」
- vpn-for-remote-work    → 「リモートワーク」「在宅勤務」「テレワーク」
- vpn-for-business-travel → 「出張」「ビジネス渡航」
- vpn-for-coworking-space → 「コワーキング」「シェアオフィス」
- fixed-ip-vpn-comparison → 「固定IP」「固定IPアドレス」

上記の語を必ず page_title と h1 に含めること。
他のトピックの語句を流用しないこと。

## タイトル・見出しの禁止語

page_title・h1 に以下を使わないこと：
- 「比較」「おすすめ」「ランキング」「最強」「No.1」「厳選」「人気」「〇選」

## 構成（この順番で固定）

1. どんな人向けか
2. どの場面で必要か
3. 選ぶ基準（判断軸）
4. 避けたい失敗・注意点
5. 確認ポイント（チェックリスト）
6. FAQ
7. CTA

## 制約
- sections は4〜6件
- checklist は5〜8件
- faq は3〜5件（回答は2〜4文）
- ブランド名・具体価格・速度数値・利用者数・稼働率を含めない
- page_title と h1 には topic_id の場面語が必ず入っていること
